5. Actions & Interaction Model

Every interactive widget can declare an action property:

KindBehaviour
navigateClient-side SPA navigation
serverPOST to a named server endpoint; receive fresh Widget IR
eventDispatches a DOM CustomEvent
copyCopy a value to clipboard

Example server action:

{
  "kind": "server",
  "name": "reindex",
  "payload": { "scope": "full" }
}

The React app POSTs to /api/widget/actions/reindex, the server executes the Goja script's actions.reindex handler and returns the updated page as Widget IR.

6. Semantic Recipes

Higher-level helpers that produce Widget IR for common patterns:

  • rag.recipes.metrics({items}) — KPI tiles in a DashboardGrid
  • rag.recipes.actionToolbar({title, actions}) — header bar with buttons
  • rag.recipes.masterDetailTable({rows, columns, detail}) — master-detail pattern
