3. Component Hierarchy

AppShell
 ├── AppNav (brand + navigation items)
 ├── DashboardGrid
 │   ├── Panel
 │   │   ├── StatusText
 │   │   ├── MetadataGrid
 │   │   └── Stack / Inline
 │   └── DataTable
 └── Panel (detail view)
      ├── DataTable (master)
      └── Panel (detail)
          └── MetadataGrid

Every UI is a tree of Widget IR nodes, serialised as JSON and rendered by the WidgetRenderer React component. Widget IR has three node kinds:

  • text – leaf text node
  • element – passthrough HTML-like element (tag, attrs, children)
  • component – typed widget node (Panel, DataTable, etc.)
