4. Borders, Borders, & Shadows

  • Border: 1px solid #000 for all components
  • Border radius: 0 (no rounding — monochrome aesthetic)
  • Shadows: None. Use borders only.
  • Focus ring: 2px inset outline, colour --mac-accent

5. Iconography

The design system deliberately does not use an icon library. Status indicators are inline SVG or Unicode characters (the StatusText component shows ✔, ●, ✘, etc.). Icon-buttons use IconButton with aria labels.
