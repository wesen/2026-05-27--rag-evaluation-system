
  # css-visual-diff - Compare rendered HTML/CSS across browser targets        
                                                                              
  Compare rendered HTML/CSS across browser targets and generate visual        
  evidence for frontend work.                                                 
                                                                              
  Start with: css-visual-diff help getting-started                            
                                                                              
  That guide walks through the smoke workflow, direct compare command, overlay
  screenshot example, review-site workflow, and the repository-scanned        
  JavaScript examples under examples/verbs.                                   
                                                                              
  To list all embedded docs, run: css-visual-diff help --list                 
                                                                              
  Run css-visual-diff help --ui to open the interactive help TUI.
