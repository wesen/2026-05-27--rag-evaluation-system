# Semantic Widget IR support for reusable design-system components

This is the document workspace for ticket RAGEVAL-WIDGET-IR-SEMANTIC-COMPONENTS.

- **design/**: Design documents and architecture notes

- **playbooks/**: Operational playbooks and procedures

- Update metadata: `docmgr meta update --ticket RAGEVAL-WIDGET-IR-SEMANTIC-COMPONENTS --field Status --value review`
