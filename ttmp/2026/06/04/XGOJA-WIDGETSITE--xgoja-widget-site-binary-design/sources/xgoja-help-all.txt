
  # xgoja - Build custom goja binaries from declarative module specs          
                                                                              
  xgoja builds custom goja-powered binaries by generating a Go program that   
  imports selected module provider packages and compiling it with the normal  
  Go toolchain.                                                               
                                                                              
  The first implementation is intentionally staged. The CLI shape is available
  first; buildspec validation and code generation are added in follow-up      
  phases.                                                                     
                                                                              
  Run xgoja help --ui to open the interactive help TUI.
